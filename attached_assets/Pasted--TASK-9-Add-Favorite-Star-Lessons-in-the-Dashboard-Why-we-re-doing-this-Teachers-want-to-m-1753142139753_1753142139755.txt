✅ STEP 3: Add a Favorite Toggle Button in the Dashboard

✅ STEP 4: Add the Toggle Route
👨‍💻 File: app.py

Add this new route below your other routes:

python
Copy
Edit
@app.route('/toggle_favorite/<int:lesson_id>', methods=['POST'])
def toggle_favorite(lesson_id):
    lesson = Lesson.query.get_or_404(lesson_id)
    lesson.is_favorite = not lesson.is_favorite
    db.session.commit()
    return redirect('/dashboard')
